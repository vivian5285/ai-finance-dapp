# 🌐 AI Finance DApp - 前端

本目录包含 **前端 Web 应用**，基于 **React + Vite** 开发，使用 **TailwindCSS** 进行样式管理。

## 🛠️ 技术栈
- React + TypeScript
- Vite（前端构建工具）
- TailwindCSS（CSS 框架）
- Web3.js（与区块链交互）

## 📂 目录结构
```bash
frontend/
│── src/
│   ├── components/   # 复用组件
│   ├── pages/        # 页面组件
│   ├── App.tsx       # 入口组件
│   ├── main.tsx      # React 挂载点
│   ├── index.css     # 全局样式
│── public/           # 静态资源
│── package.json      # 依赖管理
│── tailwind.config.js # TailwindCSS 配置
│── postcss.config.js  # PostCSS 配置
│── vite.config.ts     # Vite 配置
│── README.md          # 说明文件
